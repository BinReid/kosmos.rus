
# Космос.Rus - Российская космическая платформа 🚀

**Космос.Rus** — это образовательно-новостной портал, посвященный российской космонавтике. Платформа объединяет актуальные новости космической отрасли, интерактивные тесты для проверки знаний, систему геймификации (уровни, достижения, ежедневные серии) и профориентационный раздел для молодежи. Проект интегрирован с нейросетью **GigaChat** для автоматического сжатия новостей.

## ✨ Возможности

- **📰 Новости космонавтики:** Публикация и просмотр новостей с возможностью автоматического сжатия текста через **GigaChat API**.
- **🎓 Образовательные тесты:** Проверка знаний с начислением очков, отслеживанием прогресса и историей результатов.
- **🏆 Геймификация:** Система уровней (1-5), достижений (бронзовые, серебряные, золотые, платиновые) и ежедневных серий (streak) для мотивации пользователей.
- **🧭 Карьерный трек (16+):**
  - Информация о профильных ВУЗах (МГТУ им. Баумана, МАИ, МИФИ и др.) с описанием специальностей.
  - Описание карьерных направлений (космонавт-испытатель, инженер космических систем, оператор ЦУПа, научный сотрудник).
  - **Раздел вакансий (18+)** с актуальными предложениями от предприятий космической отрасли (РКК "Энергия", ГКНПЦ им. Хруничева и др.).
- **👤 Личный кабинет:** Аватар (загрузка или генерация), статистика, настройки профиля, смена пароля.
- **🛡️ Админ-панель:** Полное управление контентом — новости, тесты, вопросы, достижения, ВУЗы, направления карьеры, вакансии и факт дня.
- **📊 Статистика:** Просмотры новостей, активность пользователей, прогресс по уровням.
- **📱 Адаптивный дизайн:** Удобный просмотр с мобильных устройств.

## 🛠 Технологии

- **Backend:** Python 3.11+, Flask 3.1.3, SQLAlchemy 2.0.48
- **База данных:** SQLite (по умолчанию), поддерживается миграция на PostgreSQL
- **AI:** GigaChat API 0.2.0 (для суммаризации текста)
- **Формы и валидация:** Flask-WTF 1.2.2, WTForms, email-validator 2.3.0
- **Frontend:** Bootstrap 5, JavaScript (Vanilla JS + jQuery), CSS3
- **Аутентификация:** Сессии Flask, хеширование паролей (Werkzeug)

## 🚀 Установка и запуск

Следуйте этим шагам, чтобы развернуть проект локально.

### Предварительные требования

- Установленный Python версии **3.11** или выше.
- `pip` (менеджер пакетов Python).
- (Опционально) API-ключ GigaChat для работы со сжатием новостей.

### Шаг 1: Клонирование репозитория
```
git clone https://github.com/your-username/cosmos-project.git
cd cosmos-project
```
### Шаг 2: Создание виртуального окружения
```
python -m venv venv
source venv/bin/activate   # Для Linux/MacOS
venv\Scripts\activate      # Для Windows
```
### Шаг 3: Установка зависимостей
```
pip install -r requirements.txt
```
### Шаг 4: Настройка переменных окружения
Создайте файл .env в корне проекта со следующим содержимым:
```
# Секретный ключ Flask (обязательно измените в продакшене!)
SECRET_KEY=your-secret-key-here-change-in-production

# База данных (по умолчанию SQLite)
DATABASE_URL=sqlite:///cosmos.db

# API ключ GigaChat (опционально, для работы сжатия новостей)
GIGACHAT_API_KEY=your-gigachat-api-key

# Данные администратора по умолчанию
ADMIN_USERNAME=admin
ADMIN_EMAIL=admin@cosmos.ru
ADMIN_PASSWORD=admin123
```
### Шаг 5: Запуск приложения
```
python app.py
```
